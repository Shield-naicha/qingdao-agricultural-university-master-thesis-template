# 青岛农业大学硕士学位论文 LaTeX 模板（非官方）

本仓库提供一个面向**青岛农业大学硕士学位论文**排版的 LaTeX 模板。

> 说明：本模板为个人整理与开发版本，**非青岛农业大学官方发布**。正式提交前，请务必再次对照学校最新学位论文格式要求、学院补充规定和导师意见进行核查。

## 模板特点

- 支持**学术型硕士 / 专业型硕士**两套封面信息切换
- 默认采用 **GB/T 7714 著者-年份制** 参考文献样式，并支持切换为顺序编码制

## 目录结构

```text
qau-thesis-final-zhu/
├── main.tex                     % 主控文件
├── qau-thesis.sty               % 样式文件
├── qau-cover-info.tex           % 封面与扉页信息总表
├── ref.bib                      % 参考文献数据库
├── gbt7714-qau.bst              % 顺序编码制样式
├── gbt7714-qau-ay.bst           % 著者-年份制样式（默认）
├── frontmatter/                 % 封面、扉页、摘要、目录等前置部分
├── chapters/                    % 正文章节
├── appendices/                  % 附录
├── backmatter/                  % 导师组意见、致谢、简历、成果目录
├── figures/                     % 正文图片
└── assets/                      % 封面资源等
```

## 使用方法

### 1. 修改封面与英文扉页信息

优先编辑 `qau-cover-info.tex`。

该文件已经分成两部分：
- 学术型硕士参数
- 专业型硕士参数

实际使用时只保留其中一组生效，另一组整段注释掉。

### 2. 修改正文内容

- 在 `chapters/` 中按章写作
- 在 `frontmatter/abstract-cn.tex` 和 `frontmatter/abstract-en.tex` 中填写摘要
- 在 `appendices/` 中补充附录
- 在 `backmatter/` 中填写致谢、简历、科研成果等内容

### 3. 插入图片

默认图片目录为：
- `figures/`：正文插图

### 4. 管理参考文献

参考文献数据库文件为 `ref.bib`。

当前 `main.tex` 默认设置为：

```tex
\bibliographystyle{gbt7714-qau-ay}
```

这表示使用**著者-年份制**。如需切换为顺序编码制，可改为：

```tex
\bibliographystyle{gbt7714-qau}
```

## 编译方式

推荐使用 **XeLaTeX + BibTeX + XeLaTeX + XeLaTeX**。


## 关键文件说明

### `main.tex`
控制整体编排顺序，包括：
- 封面
- 英文扉页
- 声明
- 中英文摘要
- 中英文目录
- 正文章节
- 参考文献
- 附录
- 导师组意见
- 致谢
- 作者简历
- 在读期间科研学术成果目录

### `qau-thesis.sty`
主要负责：
- 页面尺寸与页边距
- 章节标题格式
- 页眉页脚
- 图表标题
- 目录缩进与引导点
- 参考文献标题和条目格式
- 附录与结尾部分标题样式

### `qau-cover-info.tex`
这是最常改的配置文件，用于统一填写：
- 论文题目
- 作者姓名
- 导师姓名
- 学位类别
- 专业领域 / 学科专业
- 学院
- 英文信息
- 提交日期
